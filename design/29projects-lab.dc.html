PowerGrid Network

"A decentralized power plant. Smart meters that settle on-chain."

Real Tapo P110 smart plugs stream live energy data through a Python oracle into four ink! smart contracts on Polkadot — device registration, grid events, token rewards, and DAO governance. Plug in, contribute energy, earn tokens. Verified end-to-end with real hardware.

Nectar Network

"One vault. Many keepers. No single point of failure."

Replaces fragile single-operator liquidation bots on Stellar's Blend Protocol with a distributed network of competing, staked keepers funded by a shared USDC vault. Depositors earn the profit from every liquidation. Staking, slashing, and Dutch-auction fills — all on Soroban.

BUILD 003 Devnet Live · Mainnet Imminent

Itô Protocol

"Math doesn't need a counterparty."

Peer-to-pool options on Solana with auto-rolling weekly vaults. Every option is priced against an on-chain Black-Scholes volatility surface — the pool itself is the counterparty. Deposit SOL, earn USDC. Built on 100 years of stochastic calculus and a 128-bit fixed-point math crate, open source from day one.

Solana Anchor / Rust Pyth oracle Fermat-Solana
